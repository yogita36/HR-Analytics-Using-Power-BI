# Strategic Diversity & Inclusion Transformation

## Overview

Welcome to the Strategic Diversity & Inclusion Transformation project. In response to the challenge of achieving gender balance at the executive management level, this initiative aims to provide comprehensive support through a strategic and holistic approach.

## Project Objectives

- **Enhancing Gender Diversity:** Addressing challenges in promoting gender diversity within the executive management team.
  
- **Business Imperatives:** Recognizing diversity and inclusion as essential business imperatives rather than optional initiatives.

## Key Components

### 1. Analytics and KPIs

This project focuses on analytics to derive meaningful insights. The following key performance indicators (KPIs) will be calculated:

- Number of Men
- Number of Women
- Number of Leavers
- Percentage of Employees Promoted (FY21)
- Percentage of Women Promoted
- Percentage of Hires: Men
- Percentage of Hires: Women
- Percentage Turnover
- Average Performance Rating: Men
- Average Performance Rating: Women
